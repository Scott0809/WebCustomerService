# WebCustomerService
web版在线客服聊天系统  
  1.支持自定制广告推送和自动答复，进入监控页面底部按钮自定制消息  
  2.开箱即用，直接启动项目通过IP:port/index进入监控页面，IP:port/customer注册并进入客服页面，IP:port/consumer注册并进入客户页面。  
    无侵入随时嵌入其他任何B/S系统。  
  3.
